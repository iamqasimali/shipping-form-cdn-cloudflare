Installation

Get started by adding the SDK script to your HTML page. It's designed to be lightweight and easy to integrate.

CDN Installation

Add this script tag to your HTML head or body:

<script src="https://shipping-form-cdn.iamqasimalizahid.workers.dev/main.js"></script>

NPM Installation Coming Soon

For module bundlers and modern JavaScript applications:

npm install shipping-form-sdk
Important: Make sure to include the SDK script before initializing the form in your JavaScript code.

Configuration

Initialize the SDK with your specific configuration options. All settings are optional except for the API endpoint.

Basic Configuration

const shippingForm = new ShippingFormSDK({
  apiEndpoint: 'https://your-api.com/api/shipments', // Required
  apiKey: 'your-api-key-here', // Required for secure APIs
  theme: 'light', // Optional: 'light' or 'dark'
  language: 'en', // Optional: Language code
  autoClose: true, // Optional: Auto-close after success
  onSuccess: function(data) {
    // Handle successful submission
    console.log('Shipping created:', data);
    alert('Success! Tracking: ' + data.tracking_number);
  },
  onError: function(error) {
    // Handle submission errors
    console.error('Error:', error);
    alert('Error: ' + error.message);
  },
  onClose: function() {
    // Handle modal close events
    console.log('Form closed by user');
  }
});

Usage Methods

Once initialized, you can control the form with these methods:

Open Form

shippingForm.open();

Close Form

shippingForm.close();

Update Configuration

shippingForm.updateConfig({
  theme: 'dark',
  apiKey: 'new-api-key'
});

Clean Up

shippingForm.destroy();

API Response Format

Your backend API should return responses in the following formats for proper SDK functionality.

Success Response (HTTP 200-299)

Return this format when submission is successful:

{
  "success": true,
  "data": {
    "id": "ship_123456",
    "tracking_number": "TRK789012",
    "status": "processing",
    "estimated_delivery": "2024-01-15",
    "shipping_cost": 24.99,
    "carrier": "UPS"
  },
  "message": "Shipping information submitted successfully"
}

Error Response (HTTP 400-599)

Return this format when there are validation errors or server issues:

{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid shipping address provided",
    "details": {
      "address": [
        "Address must be at least 10 characters long",
        "City is required"
      ],
      "zip_code": [
        "Invalid zip code format"
      ]
    }
  }
}

Required Headers

Your API should accept these headers for proper communication:

Content-Type: application/json
X-API-Key: YOUR_API_KEY
Accept: application/json
X-SDK-Version: 2.0.0

React Component Example

import React, { useEffect, useRef, useState } from 'react';

function ShippingFormButton() {
  const shippingFormRef = useRef();
  const [trackingInfo, setTrackingInfo] = useState(null);

  useEffect(() => {
    // Load SDK dynamically
    const script = document.createElement('script');
    script.src = 'https://shipping-form-cdn.iamqasimalizahid.workers.dev/main.js';
    script.onload = () => {
      shippingFormRef.current = new ShippingFormSDK({
        apiEndpoint: 'https://your-api.com/api/shipments',
        apiKey: 'your-api-key',
        onSuccess: (data) => {
          console.log('Success:', data);
          setTrackingInfo(data);
          // You can update your React state here
        },
        onError: (error) => {
          console.error('Error:', error);
          // Handle error in your React app
        }
      });
    };
    document.head.appendChild(script);

    return () => {
      // Cleanup
      if (shippingFormRef.current) {
        shippingFormRef.current.destroy();
      }
      document.head.removeChild(script);
    };
  }, []);

  const openForm = () => {
    if (shippingFormRef.current) {
      shippingFormRef.current.open();
    }
  };

  return (
    <div>
      <button 
        onClick={openForm}
        className="shipping-button"
      >
        ðŸšš Add Shipping Information
      </button>
      {trackingInfo && (
        <div className="tracking-info">
          Tracking: {trackingInfo.tracking_number}
        </div>
      )}
    </div>
  );
}

export default ShippingFormButton;
